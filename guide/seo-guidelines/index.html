5. Apply SEO Guidelines

Follow these requirements for every page to maximize search visibility.

Page Requirements

Every page on your site should include:

Element Requirement
Content Minimum 1000 words, unique and useful
Primary Keyword One per page ‚Äî in title, H1, URL, and first 100 words
External Links Minimum 3 links to high-authority sites (.gov, .edu, major publications)
FAQ Section 5+ questions and answers with FAQPage schema
Footer Embedded Google Map on every page
Schema Markup Organization + FAQPage at minimum

URL Structure

For location-based businesses, use hierarchical URLs:

/state/city/service

Examples:

  • /washington/seattle/weight-loss
  • /texas/austin/emergency-dentist
  • /california/los-angeles/teeth-whitening

Page Title Format

Primary Keyword - Secondary Keyword | Brand Name

Keep under 60 characters. Put important keywords first.

Meta Description

  • 150-160 characters
  • Include primary keyword naturally
  • Include a call-to-action
  • Unique for every page

Schema Markup

Include JSON-LD schema for:

  • Organization ‚Äî every site
  • LocalBusiness ‚Äî for businesses with physical locations
  • FAQPage ‚Äî every page (5+ Q&As)
  • BreadcrumbList ‚Äî for navigation

Pre-Launch Checklist

Before any page goes live:

  • ‚òê Keyword map documented
  • ‚òê Unique title and meta description
  • ‚òê H1 contains primary keyword
  • ‚òê Primary keyword in first 100 words
  • ‚òê 1000+ words of content
  • ‚òê 3+ external links to authority sites
  • ‚òê 5+ FAQ Q&As with schema
  • ‚òê Images have alt text
  • ‚òê Internal links to related pages
  • ‚òê Google Map in footer
  • ‚òê Schema markup validated
  • ‚òê Mobile-friendly
  • ‚òê Page speed optimized
